In order to run this tournament, you will need to:

1. Install Vagrant and VirtualBox
2. Install PostgreSQL on your Vagrant VM
3. Use the command 'vagrant up' to power it on, then log in with 'vagrant ssh'
4. Build and access the tournament database by running 'psql' then '\i tournament.sql'
5. Exit from psql with '\q'
6. Call up desired Python functions
